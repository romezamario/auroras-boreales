Uso de geolocalización basada en dirección IP

← Volver al mapa principal

El sistema implementado incorpora una funcionalidad de geolocalización aproximada del usuario con el objetivo de contextualizar visualmente la información mostrada en un mapa dentro de la interfaz de la aplicación web. Para ello, al momento de realizar una consulta al sitio, se utiliza la dirección IP desde la cual se origina la solicitud HTTP.

La dirección IP se consulta de manera temporal mediante el servicio externo disponible en https://ipapi.co/json/, el cual permite obtener información geográfica aproximada asociada a dicha dirección, incluyendo latitud, longitud, país y región administrativa. Esta información es utilizada exclusivamente para ubicar un marcador en el mapa interactivo que permite al usuario visualizar su posición aproximada en relación con los datos geográficos presentados por la aplicación.

Es importante señalar que la dirección IP no es almacenada ni persistida por el sistema desarrollado, ni se asocia a identificadores personales del usuario. El procesamiento de esta información se realiza únicamente durante el tiempo necesario para generar la visualización geográfica en la sesión activa del usuario.

Desde una perspectiva regulatoria, la dirección IP puede ser considerada un dato personal en determinados marcos normativos, como el Reglamento General de Protección de Datos (GDPR) de la Unión Europea, en la medida en que potencialmente podría permitir la identificación indirecta de un individuo. No obstante, en el contexto de este sistema el uso de la dirección IP se limita a un procesamiento efímero con fines estrictamente funcionales, sin almacenamiento ni elaboración de perfiles, lo que reduce significativamente los riesgos asociados al tratamiento de datos personales.

La geolocalización obtenida mediante este método es aproximada y depende de las bases de datos de asignación de direcciones IP mantenidas por el proveedor del servicio. En consecuencia, la ubicación mostrada debe interpretarse únicamente como una estimación de la localización geográfica del usuario y no como una posición exacta.
